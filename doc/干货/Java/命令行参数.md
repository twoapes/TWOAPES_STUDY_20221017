# 命令行参数

```shell
java [ options ] class [ arguments ]
```

例子

```shell
javac CommandLine.java
java CommandLine 1 2 3
```

```java
package grammar;

/**
 * @author huyingzhao
 * 2022-01-02 14:01
 */
public class CommandLine {
    public static void main(String[] args) {
        for (String string : args) {
            System.out.println(string);
        }
    }
}
```
