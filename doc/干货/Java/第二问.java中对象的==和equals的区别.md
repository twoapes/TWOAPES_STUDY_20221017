# 第二问:java中对象的==和equals的区别

```java
/**
 * @author add by huyingzhao
 * 2022-06-12 18:34
 * 变量-第二问:java中对象的==和equals的区别
 */
@Slf4j
public class Extend2 {
    public static void extend2() {
        StringBuilder stringBuilder = new StringBuilder("\n");
        //操作符 ==
        //基本数据类型:比较的是值
        int a = 10;
        int b = 10;
        stringBuilder.append("基本数据类型:").append(a == b).append("\n");
        //引用数据类型,基本类型的包装类:比较的是内存地址
        Integer integer = 110;
        Integer integer2 = 110;
        stringBuilder.append("引用数据类型内存地址相同:").append(integer == integer2).append("\n");

        integer = 128;
        integer2 = 128;
        stringBuilder.append("引用数据类型内存地址不同").append(integer == integer2).append("\n");

        //equals方法
        //未重写equals方法的类：效果和==一样
        //重写了equals方法的类：会重新定义判断等值逻辑,使其更符合其类本身的比较方法
        //例如String和Integer,为了能对比他们的值不同,因而都重写了equals 即我们一般只想知道他们的内容是否一致,而不会关心地址是否一样
        integer = 128;
        integer2 = 128;
        stringBuilder.append("Integer重写equals方法的内容比较结果:").append(integer.equals(integer2)).append("\n");

        String str1 = "128";
        String str2 = "128";
        stringBuilder.append("String重写equals方法的内容比较结果:").append(str1.equals(str2)).append("\n");
        log.info(stringBuilder.toString());
    }
}
```
