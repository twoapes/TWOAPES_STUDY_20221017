# 包装类

- 针对八种基本数据类型定义相应的引用类型—包装类（封装类)

- 有了类的特点，就可以调用类中的方法，因为Java面向对象的

![20230715214255](https://twoapes.oss-cn-shenzhen.aliyuncs.com/image/20230715214255.png)

## 装箱

基本数据类型包装成包装类的实例

- 包装类的构造器实现

```java
Integer i=new  Integer();
```

- 字符串参数构造包装类

```java
Float f=new Float("3.14");
```

## 拆箱

获得包装类对象中的基本数据类型变量

- 调用包装类的.xxxValue()方法

```java
boolean b=obj.booleanValue();
```

## 字符串转换成基本数据类型

```java
int i=new Integer("12");
```

- 包装类的parseXxx(String s)静态方法

```java
Float f=Float.parseFloat("3.14");jav
```

## 基本数据类型转换成字符串

- 调用字符串重载的valueOf()方法

```java
String str=String.valueOf(3.14f);
```

- 更直接的方式

```java
String str=5+"";
```

jdk1.5之后,支持自动装箱和自动拆箱,但是类型必须匹配

以下是一个演示Java包装类的示例，其中涉及到所有的基本类型对应的包装类：

```java
public class WrapperDemo {
    public static void main(String[] args) {
        // Boolean wrapper class example
        Boolean boolObj = Boolean.valueOf(true);
        System.out.println("Boolean wrapper class example: " + boolObj);

        // Byte wrapper class example
        Byte byteObj = Byte.valueOf((byte) 10);
        System.out.println("Byte wrapper class example: " + byteObj);

        // Short wrapper class example
        Short shortObj = Short.valueOf((short) 20);
        System.out.println("Short wrapper class example: " + shortObj);

        // Integer wrapper class example
        Integer intObj = Integer.valueOf(30);
        System.out.println("Integer wrapper class example: " + intObj);

        // Long wrapper class example
        Long longObj = Long.valueOf(40L);
        System.out.println("Long wrapper class example: " + longObj);

        // Float wrapper class example
        Float floatObj = Float.valueOf(50.5f);
        System.out.println("Float wrapper class example: " + floatObj);

        // Double wrapper class example
        Double doubleObj = Double.valueOf(60.6);
        System.out.println("Double wrapper class example: " + doubleObj);

        // Character wrapper class example
        Character charObj = Character.valueOf('a');
        System.out.println("Character wrapper class example: " + charObj);
    }
}
```

在这个例子中，我们使用了所有基本类型的包装类，通过调用valueOf方法将基本类型的值转换为对应的包装类型。我们还打印了每个包装类的实例，以演示它们的用法。
